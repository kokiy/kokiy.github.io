# linux 常用命令

| 命令                                           | 说明                 |
| ---------------------------------------------- | -------------------- |
| `ssh-copy-id -i /root/.ssh/id_rsa.pub root@ip` | 本地 ssh 推送到远端  |
| `ssh root@ip`                                  | ssh 远端登录         |
| `logout`                                       | ssh 退出登录         |
| `mv a.name b.name`                             | 修改文件名字         |
| `scp car.txt root@ip:/home`                    | 本地文件推送到 linux |
| `scp root@ip:/home/car.txt /`                  | 远端文件下载到本地   |

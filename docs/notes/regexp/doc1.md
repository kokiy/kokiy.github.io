# 正则规则

| 表达式   | 说明                                  |
| -------- | ------------------------------------- |
| `[0-9]`  | 匹配 0-9 范围内的任意数字             |
| `[02az]` | 只匹配列举出来的值 0,2,a,z            |
| `\d`     | 和[0-9]一样的效果,匹配任意阿拉伯数字  |
| `\D`     | 匹配任意一个非数字字符                |
| `{m}`    | m 次                                  |
| `{m,n}`  | m 次到 n 次                           |
| `.`      | 匹配任意字符                          |
| `?`      | 0 个或者 1 个                         |
| `*`      | 0 个或者多个                          |
| `+`      | 1 个或者多个                          |
| `[^123]` | 取反,排除 123                         |
| `\w`     | 单词字符,字母数字下划线`[_a-zA-Z0-9]` |
| `\W`     | 非单词字符,空格\|标点符号等字符       |
| `\s`     | 空白符 == `[ \t\n\r]`                 |
| `\S`     | 非空白符                              |
| `\t`     | 制表符                                |
| `\n`     | 换行符                                |
| `\r`     | 回车符                                |
| `\b`     | 匹配单词边界                          |

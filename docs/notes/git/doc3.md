# 提交规范

| 语法        | 说明                                  |
| ----------- | ------------------------------------- |
| `feat:`     | 新功能                                |
| `fix:`      | 修复 bug                              |
| `doc:`      | 文档改变                              |
| `style:`    | 代码格式改变                          |
| `refactor:` | 某个已有功能重构                      |
| `perf:`     | 性能优化                              |
| `test:`     | 增加测试                              |
| `build:`    | 改变了 build 工具 如 grunt 换成了 npm |
| `revert:`   | 撤销上一次的 commit                   |
